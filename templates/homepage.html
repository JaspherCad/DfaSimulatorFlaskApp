Topic 2: Usage of DFA

Starting from the leftmost part of the string and an initial state.

  • Scan one symbol from string
  • From current state

How to use an DFA?

  1. Write down the initial state
  2. Scan one symbol at a time from left to right.
  3. Using current state and current symbol, determine new state
  4. Repeat from step 2, until end of string is reached.

Possible Output

  1. Accepted: string is recognized by the machine
  2. Rejected: string is not recognized by the machine

Accepting Conditions

  1. Scan the whole input string
  2. Land on a final state

Example

Determine if the following strings are accepted by using machine M1

  1. 10000000

    q1,1 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4

    Accepted

  2. 01111

    q1,0 -> q2,1 -> q2,1 -> q2,1 -> q2,1 -> q2

    Accepted

  3. 1000001

    q1,1 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4,0 -> q4,1 -> q3

    Rejected q3 is not a final state
